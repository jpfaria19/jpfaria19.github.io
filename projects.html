Meus projetos

Abaixo você vai encontrar alguns projetos que desenvolvi(ou venho desenvolvendo), o código-fonte está disponível no meu github.

WeCoin

WeCoin é um projeto ambicioso, desenvolvido com um grupo de amigos da faculdade. Seu objetivo é unir amantes, especialistas e todo tipo de pessoa que se interessa por criptomoedas em uma rede social. Infelizmente hoje o site encontra-se em manutenção e por este motivo há uma certa lentidão mas estamos trabalhando para resolver esse problema. (Infelizmente o código-fonte do WeCoin não está publico.)

Guia Rio

Projeto final do período de Front-End. Guia Rio é um site voltado para turistas e viajantes que buscam uma ajuda para se localizar no Rio de Janeiro. Lá eles podem encontrar um restaurante para jantar, como chegar em um ponto turístico, hotéis e muito mais.

Memory Game - Joker

Desenvolvi esse jogo para o projeto final de JavaScritp (dentro do período de Front-End). É um jogo da memória com cartas de baralho, no final ele mostra quantas jogadas você fez e o tempo que levou para completar o jogo.

Boletim Escolar

Um dos meus projetos é o boletim escolar, ele faz o papel de gerenciador onde você adiciona quantos alunos quiser, coloca suas notas e ele faz a média, ainda tem um filtro para uma procura melhor.
